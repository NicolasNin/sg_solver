# Star Genius Solver
An automated puzzle solver for Star Genius, designed to find optimal solutions using efficient search algorithms.
## Features
- **Image Recognition**: Automatically detect board states from photos.
- **Web Interface**: A small web application for testing and visualization.

## how to use
1. Clone the repository to your local machine.
2. Install the necessary dependencies.
3. In app directory, run `python server.py`.
